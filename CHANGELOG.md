# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-18

### Added

- Initial release of the portfolio
- 3D interactive hero section with Three.js
- Responsive glassmorphism UI design
- Lazy-loaded components for optimal performance
- Project carousel with spotlight hover effects
- Terminal-themed contact section
- Animated skill grid with progress indicators
- Experience timeline with animated entries
- Theme switching (dark/light/system)
- Comprehensive accessibility features (WCAG AAA)
- Custom animated gradient utilities
- Framer Motion animations throughout
- Optimized bundle splitting strategy
- Mobile-responsive design
- Keyboard navigation support
- Reduced motion preferences support

### Performance

- Code splitting with React.lazy()
- Manual vendor chunking (React, Three.js, Radix UI)
- 3D rendering optimizations (30 FPS cap)
- Intersection observer for off-screen pausing
- Bundle size optimized (<1MB per chunk)

### Accessibility

- WCAG AAA contrast ratios
- Full keyboard navigation
- Screen reader compatibility
- ARIA labels and roles
- Focus indicators
- Semantic HTML structure
