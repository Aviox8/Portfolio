## Color Palette Reference

### Primary Colors
```
Cyan:       #00ffff  (RGB: 0, 255, 255)   - Primary accents
Violet:     #aa00ff  (RGB: 170, 0, 255)   - Secondary accents
White:      #ffffff  (RGB: 255, 255, 255) - Text primary
```

### Background Colors
```
Void Black:     #000000  - Main background
Starry Indigo:  #0f0f23  - Secondary bg
Cosmic Purple:  #1a0a33  - Tertiary bg
Muted Purple:   #2a0a2e  - Borders
```

### Text Colors
```
Primary Text:   #ffffff       - Headers, main content
Muted Text:     #b0b0b0       - Secondary content
Disabled Text:  #666666       - Disabled states
```

---

## Glow Effects Hierarchy

```
1. Intense Glow (Hover):
   box-shadow: 0 0 40px rgba(0, 255, 255, 0.8)

2. Normal Glow (Default):
   box-shadow: 0 0 20px rgba(0, 255, 255, 0.5)

3. Subtle Glow (Background):
   box-shadow: 0 0 10px rgba(0, 255, 255, 0.3)

4. Text Glow:
   text-shadow: 0 0 20px rgba(0, 255, 255, 0.6),
                0 0 40px rgba(170, 0, 255, 0.4)
```

---

## Animation Timing

```
Fast:         0.2s   (Menu toggle, icon rotation)
Default:      0.3s   (Hover effects, transitions)
Medium:       0.5s   (Section reveals, hero text)
Slow:         0.8s   (Parallax, scroll effects)
Very Slow:   3.0s+   (Breathing effects, float loops)

Spring physics for lift/float effects:
  stiffness: 300-400
  damping: 20-25
```

## Device Variants

### Desktop (1200px+)
- Full particle system (18 particles)
- Icon navigation
- Cursor glow effect
- 3D hero sphere
- Full animations

### Tablet (640px-1200px)
- Reduced particles (12)
- Adjusted spacing
- Simplified animations
- Touch-optimized buttons
- Sidebar for nav

### Mobile (<640px)
- Minimal particles (8)
- Icon + menu toggle
- Slide-in sidebar
- Reduced 3D effects
- Optimized touch targets (44x44px min)

## Accessibility Enhancements

### High Contrast Mode (cosmic-high)
```
Enabled by: [data-theme="cosmic"][data-contrast="high"]

Changes:
- Cyan: #00ffff â†’ #00ffff (unchanged, maximum brightness)
- Violet: #aa00ff â†’ #ff00ff (magenta, higher contrast)
- Increased glow intensity
- Enhanced text shadows
- More prominent focus indicators
```

### Focus States
```
All interactive elements show:
- 2px solid cyan outline
- Cyan glow shadow
- Keyboard-visible indicators
- No color-only differentiation
```

### Reduced Motion
```
When prefers-reduced-motion is set:
- Disable floating particle system
- Disable parallax effects
- Remove distortion waves
- Keep essential transitions (0.2s)
- Maintain accessibility animations
```
